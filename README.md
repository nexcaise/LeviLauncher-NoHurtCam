# LeviLauncher - NoHurtCam

> Only works on versions below or equal to 1.21.120

# About
Disables a built-in Minecraft feature which allows the camera to shake when taking damage.

* LeviLauncher(android) â€” <https://github.com/LiteLDev/LeviLaunchroid>
* libNoHurtCam mod installed

## Installation

1. Download libNoHurtCam from [Releases](https://github.com/NexCaise/LeviLauncher-NoHurtCam/releases)
2. Install LeviLauncher
3. Add libNoHurtCam mod in LeviLauncher
4. Launch Minecraft through LeviLauncher
